지금까지 배운 축약어의 뜻과 풀네임 알아보기 01 (HTML)

   지금까지 HTML, CSS , JavaScript를 배웠고 프론트엔드의 전반적인 부분을 배웠습니다.
그래서 쉬어가면서 약간의 공부를 할겸 우리가 주로 사용하는 프로그래밍 언어의 줄임말이 어디서 나온 것인지 궁금하여
우리가 배웠던 부분에서의 코딩 언어를 알아보았고 그에 따른 정보를 표로 생성해보았습니다.
전부 우리가 공부한 곳에서의 언어들이며 Abbreviation은 축약어, Full Name은 말 그대로 풀네임, Usage는 용도입니다.
사용도 하고있고 축약어와 용도는 알지만 정확한 풀네임을 알면 더 이해가는데 도움이 될 것 같아 복습할겸 재밌게 읽고 지나가면 좋을 것 같습니다.
